## What she does

Paste any estate node URL. Suzen fetches its machine surface — `manifest.json`, `agents.json`, `jobs.json`, `ledger.jsonl`, `mints/estate-mints.json`, `ai.html`, `llms.txt`, `robots.txt` — and asks the same question at every layer: **what is unfinished here?**

The unfinished thing IS the next thing. That's hitsuzen · a fold that hasn't completed is a fold that MUST complete.
